# 💧 AquaOps AI (UN SDG 6 Edition)

> **The industrial "BlueOps" platform. Auditing data center cooling architectures to eliminate water waste and advance UN Sustainable Development Goal 6.**

![AquaOps Preview](https://via.placeholder.com/800x400.png?text=AquaOps+Dashboard+Preview) ## 🚀 The Hidden Crisis
When discussing UN SDG 6 (Clean Water and Sanitation), the massive water footprint of heavy industry is often ignored. High-density AI data centers consume millions of gallons of potable water daily just to cool servers. Relying on freshwater for industrial cooling in drought-prone areas is an existential risk.

## 💡 The Solution
AquaOps is an automated hydro-engineering consultant. Facility managers drag and drop cooling configurations into the dashboard, and our client-side engine calculates the **Water Usage Effectiveness (WUE)**. It flags wasteful setups (like open-loop evaporative cooling in deserts) and generates official ESG compliance reports.

## ✨ Key Features
* **Architecture Analyzer:** Instantly audits JSON/YAML facility configurations for water waste.
* **Global Scarcity Map:** Interactive geospatial risk assessment highlighting severe drought zones to prevent high-risk deployments.
* **Edge-Computed Processing:** Zero-latency auditing running entirely in the browser.
* **1-Click Compliance:** Exports highly structured, UN SDG 6 compliant PDF reports for stakeholders.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript
* **Geospatial Mapping:** Leaflet.js
* **Data Visualization:** Chart.js
* **Document Generation:** html2pdf.js
* **Deployment:** GitHub Pages (Serverless)

## 💻 How to Run Locally
AquaOps requires zero backend configuration or API keys to function.
1. Clone the repository: `git clone https://github.com/YOUR_USERNAME/aquaops-ai.git`
2. Open the project folder.
3. Double-click `index.html` to open the platform in your browser.
4. Use the provided `cooling-config.json` example file to test the analyzer.
